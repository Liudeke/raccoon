ACInterface2DMultiPhase1

Gradient energy Allen-Cahn Kernel where the derivative of interface parameter kappa wrt the gradient of order parameter is considered.

Implements the Allen-Cahn term for the gradient energy contribution for the anisotropic interface energy case. Its weak form is where (kappa_name) is the gradient energy coefficent, the non-conserved non-linear order parameter variable the kernel is acting on, (mob_name) is the scalar (isotropic) mobility associated with the order parameter, and is the test function.

Input Parameters

  • etasAll other coupled order parameters eta_i of the multiphase problem

    C++ Type:std::vector<VariableName>

    Controllable:No

    Description:All other coupled order parameters eta_i of the multiphase problem

  • variableThe name of the variable that this residual object operates on

    C++ Type:NonlinearVariableName

    Controllable:No

    Description:The name of the variable that this residual object operates on
