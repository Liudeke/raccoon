PolarMomentOfInertia

Compute the polar moment of inertia of a sideset w.r.t. a point and a direction

Overview

This postprocessor computes the polar moment of inertia (or ) of a surface with respect to a reference point (origin) and an axis of rotation (direction). Its result is needed by the Torque boundary condition.

Example Input File Syntax

Input Parameters

  • boundaryThe list of boundary IDs from the mesh where this boundary condition applies

    C++ Type:std::vector<BoundaryName>

    Controllable:No

    Description:The list of boundary IDs from the mesh where this boundary condition applies

  • directionAxis direction

    C++ Type:libMesh::VectorValue<double>

    Controllable:No

    Description:Axis direction

  • originAxis origin

    C++ Type:libMesh::Point

    Controllable:No

    Description:Axis origin

    Default:TIMESTEP_END

    C++ Type:ExecFlagEnum

    Options:NONE, INITIAL, LINEAR, NONLINEAR, TIMESTEP_END, TIMESTEP_BEGIN, FINAL, CUSTOM, TRANSFER, ALWAYS

    Controllable:No

    Description:The list of flag(s) indicating when this object should be executed, the available options include NONE, INITIAL, LINEAR, NONLINEAR, TIMESTEP_END, TIMESTEP_BEGIN, FINAL, CUSTOM, ALWAYS.
